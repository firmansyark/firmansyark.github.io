Windows 10 gak pake lemot?

30 December 2019 Anggi Firmansyah

tak perlu khawatir soal memory, lemot, update.

Sedari 2018, aku mulai beralih ke Windows 10 mod dengan embel-embel LITE (selanjutnya disebut winmod). Dimana file system windows yang tak diperlukan dihilangkan dan mengutamakan file-file system yang diperlukan saja. Misalnya system yang tak diperlukan seperti CORTANA. Yah alasan utamanya karena spek laptop uda ga kuat dengan Windows 10 update'an terbaru.. *kenapa ga install win7/8 aja? | laptopku hanya support win10

Awal mula

Dari awalnya yang nemuin winmod berukuran mulai dari 700MB hingga 2GB dengan tampilan yang kece, tapi tetep mereka punya kekurangan yang sangat mengganggu kelangsungan develop maupun editing.

Winmod 700MB

Misal, winmod 700MB ini sangat ringan, tapi file system semua dibuang tanpa pertimbangan. Untuk render editing sangat membantu, tapi tidak untuk develop yg software'nya memerlukan beberapa file system windows.

Winmod 2GB

Lalu untuk winmod 2GB ini lumayan stabil, kegiatan develop dapat berjalan lancar, tampilan kece. Tapi dari segi editing sangat diberatkan dengan tampilan yang mereka mod.

Akhir mula

Dan akhirnya di 2019, nemuin winmod versi Oprekin. Windows 10 mod dengan dua varian, yaitu Lite dan Lite Plus. Meski ini oprekan, namun versi OSnya adalah update'an yang terbaru. tidak ada auto update dalam Operkin ini, jadi tak perlu khawatir dipaksa Windows untuk update OS. Setelah instal juga tak perlu activator karena sudah otomatis diaktifasi. berikut perbedaan antara Lite dan Lite Plus:

Lite
  • Berukuran 1GB
  • Untuk pengguna yang mengutamakan fungsi penting (advanced)
  • Dihilangakan:
    • Semua app bloatware (seperti Calculator, Paint, Games)
    • Windows firewall (GUI)
    • Windows defender
    • Windows update
    • Cortona
    • dsj
Lite Plus
  • Berukuran 2GB
  • Untuk pengguna umum
  • Dihilangkan:
    • Windows defender
    • Windows firewall (GUI only)
    • Windows Home Group
    • dsj

winmod Oprekin tersedia versi Pro dan Enterprise‘nya. setelah mencoba meng'instal versi Lite, kini kinerja laptop semakin lanjay tanpa terganggu paksaan update windows. Dan dengan nomor versinya yang terbaru tak perlu khawatir lagi akan software yang memerlukan requirement windows update'an terbaru, seperti keluarga Adobe CC 2019.
